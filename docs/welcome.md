# MD TUI - Welcome

## Welcome to MD TUI

MD TUI is a Markdown viewer for the terminal with support for **Mermaid** and **D2** diagrams.

### Quick Start

1. **Browse Files**: Navigate your filesystem using the file browser
2. **Open Markdown**: Press Enter on any `.md` file to view it
3. **View Diagrams**: Press `v` when viewing a markdown file with diagram blocks

### Features

- **Touch-friendly**: Large buttons for mobile/tablet use
- **Diagram Support**: Mermaid and D2 diagram rendering
- **Keyboard Shortcuts**: Full keyboard navigation support
- **Cross-platform**: Works on Linux, Android (Termux), and macOS

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| ↑/↓ | Navigate up/down |
| Enter | Open file/folder |
| Backspace | Go to parent directory |
| q | Quit / Go back |
| h or ? | Show help |
| r | Refresh file list |

### For More Information

Press `h` or `?` anytime to access the full help guide.
